console.log is streaming..

Are you ready to learn something new?

  1. Open your browsers console. (Cmd + Alt + K in Firefox, Cmd + Alt + I in Chrome)
  2. You will see some logging output already created by this demo.
  3. Now it is your turn. Try a simple command like: 
    console.log("This is my personal cloudlog hello world!")
  4. Or did you know, that there are a lot more console features? Try something more complex: 
    console.table({randomNumber: Math.random(), userAgent: navigator.userAgent})
  5. Go back the cloudlog user interface to see your streamed logs.
